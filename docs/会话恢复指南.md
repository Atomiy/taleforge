## 项目概览
- **项目**: TaleForge — AI 智能故事生成器
- **技术栈**: 后端 FastAPI (Python3)，前端 Vue 3 (CDN) + Tailwind CSS
- **运行方式**: `python -m uvicorn backend.main:app --host 0.0.0.0 --port 8080`
- **当前目录**: `c:\Users\33173\Desktop\study\taleforge`

## 当前状态（v0.4.0）
### 已完成的核心功能
1. **作品/卷/章节三级结构** — Work → Volume → Story 层级存储
2. **卷规划系统** — 每卷可设定：大纲、职责、卷间衔接、出场人物、伏笔清单
3. **三章节生成验证通过** — 第三部作品「深渊回响」（克苏鲁+硬科幻）三章连贯性评分 9.0/10
4. **灵感 API** — `POST /api/v1/story/inspire` 自动生成剧情点子/伏笔/角色建议
5. **前端交互** — 创作页、灵感弹窗、续写下一章按钮、Toast 通知、键盘快捷键
6. **章节续写三层控制** — 战术层（首段200字强制过渡）+ 战略层（大纲最高优先级）+ 自由层（风格参考）
7. **进度锚点规则** — 新章节禁止时间回溯/场景重置，起始状态必须 >= 上章末尾
