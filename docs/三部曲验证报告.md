# 第1章 狼灵的低语

寒风裹挟着雪粒，如刀锋般抽打在脸上。北风跪在祭坛前的冻土上，膝盖传来的刺痛远不及胸腔里那团灼热。狼神祭坛的图腾柱在晨光中泛着幽蓝，石壁上密密麻麻的狼爪印，像是千百年来无数先祖的目光，正冷冷地注视着他。

“霜狼部落的少年，以血为契，以骨为誓——”萨满苍老的声音在祭坛上空回荡，手中的骨刀在火盆上烤得发红，“你愿将灵魂献给狼神，成为永不退缩的战士吗？”

北风抬起头，琥珀色的眼眸对上萨满浑浊的双眼。他张了张嘴，喉咙却像被冻住了一般，发不出任何声音。昨夜的梦境还在脑海里翻涌——一头银白色的巨狼站在月光下，琥珀色的眼睛和他一模一样。它低垂着头，像是在哀求什么。

“北风！”人群中传来父亲曾经的副手铁骨的怒吼，“别像个娘们似的磨蹭！”

身后的窃窃私语如冰锥般刺来。北风咬紧牙关，用力点头。萨满的骨刀划破他的掌心，鲜血滴落在祭坛中央的狼首图腾上。

那一瞬间，世界突然安静了。

风声、火声、人群的呼吸声全部消失，只剩下一种古老的低语从地底深处涌起，像无数头狼在同时嗥叫。北风感到胸腔里的那团灼热猛地炸开，一股不属于他的记忆如洪水般冲入脑海——

他看到父亲霜牙站在同样的祭坛上，年轻的脸上满是骄傲；他看到父亲跪在冰原上，抱着一头奄奄一息的银色母狼痛哭；他看到父亲转身离去时回望部落的最后一眼，那只银色的左眼里全是绝望。

“不——”北风想要喊出声，却发现自己的声音变成了低沉的狼嚎。

祭坛上的图腾柱突然裂开，幽蓝的光芒如活物般涌出，缠绕住北风的身体。萨满惊恐地后退，骨刀掉在地上发出脆响。人群中爆发出惊呼，几个年长的猎手已经拔出了弯刀。

“狼灵！狼灵在他体内苏醒了！”萨满的声音里带着颤抖，“这是……这是霜狼血脉的觉醒！”

北风感到自己的身体在变化——指尖长出利爪，脊背弓起，皮肤下仿佛有无数条蛇在游走。他想要控制这股力量，却发现自己像溺水的人一样无力。那些不属于他的记忆还在涌入：父亲在月光下与狼群共舞，父亲用身体挡住猎手的箭矢，父亲在暴风雪中为刚出生的狼崽取暖。

“杀了他！”铁骨的声音像炸雷般响起，“他会变成狼人，会背叛部落！”

弯刀出鞘的声音此起彼伏。北风抬起头，看到族人们惊恐而愤怒的脸。他们的眼睛里没有一丝怜悯，只有对异类的恐惧。他想要解释，想要告诉他们自己还是北风，还是那个从小和他们一起长大的少年，可嘴里发出的只有狼嗥。

就在这时，一声更加悠长的狼嚎从远处的冰原传来。那声音穿透风雪，带着说不清的悲伤和警告。北风的心脏猛地一抽——那是父亲的声音，他不会认错。

“霜牙！”铁骨的脸色变得铁青，“果然是他搞的鬼！追！”

猎手们向声音传来的方向冲去，北风却瘫倒在祭坛上。狼灵的力量还在体内翻涌，但他不再抵抗了。因为他终于明白，父亲当年为什么会选择背叛部落——那些记忆里的银狼，那双哀求的眼睛，那种被误解的孤独，此刻都变成了他身体里最真实的疼痛。

他闭上眼睛，任由泪水滑落。在意识彻底模糊之前，他听到了父亲的声音，低沉沙哑，如碎石摩擦：“活下去，北风……找到月牙谷，那里有答案。”

风雪更大了。祭坛上的狼首图腾还在散发着幽蓝的光芒，像是在回应着什么。远处的冰原上，猎手们的火把渐渐消失在暴风雪中，只剩下北风一个人跪在冰冷的冻土上。身体里的狼灵还在低声嗥叫，像是要把他整个人撕碎。

# 第2章 霜狼之怒

铁山伫立在部落议事厅外的冻土上，灰色的眼眸如鹰隼般死死锁住前方。

炎龙帝国的先锋部队已推进至村口。赤焰骑在一匹披甲战马上，手中龙牙长剑斜指苍穹。他身后是两百名全副武装的帝国铁骑，铠甲在风雪中泛着暗红色的微光，如同一片移动的血海，正缓缓吞噬着这片雪原。

“交出狼神之子，否则屠尽霜狼部落。”赤焰的声音不高，却如利刃般穿透呼啸的北风，“这是帝国先知的预言——狼神之子觉醒之日，便是炎龙帝国覆灭之时。我奉皇帝之命，必须在预言实现之前将其扼杀。”

铁山的心脏狠狠一抽。

他们怎么知道的？狼神之力在他体内觉醒不过三天，连部落里的长老都尚未完全确认，帝国先知怎么可能……

“我不知道你在说什么。”铁山咬牙开口，声音沙哑如磨石，“霜狼部落没有什么狼神之子。”

“是吗？”赤焰冷笑，从怀中掏出一块泛着红光的晶石，“帝国先知用龙血占卜，狼神之子的气息就在你们部落。你以为能藏得住？”

晶石骤然剧烈震动，红光如鲜血般涌向铁山。

铁山体内的狼灵瞬间暴动，一股狂暴的力量从骨髓深处炸裂开来。他的瞳孔急剧收缩，灰色眼眸转为幽蓝，手指不自觉地弯曲成爪，指甲变长、变尖，刺破掌心。

“果然是你。”赤焰眼中闪过一丝兴奋，“动手！”

两百名铁骑同时冲锋，马蹄踏碎冻土，长矛如林般刺来。

“铁山，退后！”部落长老的喊声从身后传来。

但铁山没有退。

狼灵在体内咆哮，那股力量太强了，强到他的意识开始模糊。他能感觉到牙齿在变长，骨头在嘎吱作响，皮肤下有什么东西在疯狂生长，撕扯着他的血肉。

“吼——”

一声震彻天地的狼嚎从他喉咙里爆发出来，震得雪花簌簌落下。

铁山的身体开始变形。骨骼扭曲重组，肌肉膨胀撕裂衣物，脊背弓起如满月。黑色的狼毛从皮肤下钻出，双手变成利爪，整个人在风雪中急剧膨胀，仿佛大地在隆起。

当尘埃落定时，一头足有三丈高的巨大霜狼矗立在部落门口。

它通体银白，双眼燃烧着幽蓝的火焰，每一次呼吸都在空气中凝成冰晶，如雾如霜。狼首低垂，獠牙外露，喉咙里发出低沉的咆哮声，震得大地都在颤抖。

“狼神……”长老跪倒在地，老泪纵横。

赤焰的脸色终于变了。

“列阵！龙鳞盾！”

帝国铁骑迅速组成防御阵型，赤红色的盾牌连成一片，如一条盘踞的巨龙，鳞片在风雪中闪烁着危险的光芒。

但霜狼已经动了。

它一跃而起，三丈高的身躯如雪崩般砸入敌阵。利爪挥下，五名骑兵连人带马被撕成碎片，鲜血在雪地上炸开，如盛开的红莲。巨口张开，一口咬碎了三面龙鳞盾，铁片在齿间碎裂，发出刺耳的金属哀鸣。

铁山已经听不见任何声音了。

兽性在血液里沸腾，杀戮的本能淹没了一切。他能闻到敌人的恐惧，那是一种酸涩的气味；能听到骨头的碎裂声，清脆如折断的枯枝；能尝到铁锈般的血腥味，在舌尖上蔓延。这种感觉太美妙了，美妙到让他想要永远沉沦下去，永远做一头只知道杀戮的野兽。

“撤退！快撤退！”赤焰的喊声在远处响起，带着一丝不易察觉的颤抖。

但霜狼怎么可能放过猎物？

它追着溃散的骑兵冲入风雪，一口一个，将逃窜的敌人撕成碎块。鲜血染红了白雪，残肢断臂散落一地，整个战场变成了屠宰场，空气中弥漫着浓重的血腥味。

就在这时，铁山看到了一个身影。

那是一个穿着狼皮的中年男人，站在远处的冰崖上，身影在风雪中若隐若现。他的眼睛是幽蓝色的，和铁山此刻的眼睛一模一样，仿佛两团燃烧的鬼火。

父亲……

铁山的意识猛地一颤。

那个男人转身向冰原深处跑去，动作轻盈得像一头真正的狼，每一步都踏在雪上不留痕迹。他的方向很明确——月牙谷。

“北风……找到月牙谷……”

铁山想起了父亲留下的那句话。他想追上去，但身体已经不受控制了。霜狼的兽性重新占据上风，它调转方向，向部落冲去，眼中只有杀戮的欲望。

“铁山！是我们！”长老的声音在风中颤抖，苍老的声音里满是恐惧，“醒醒！”

但霜狼的眼睛里只有杀戮。

它扑向部落的木栅栏，一口咬碎，木屑四溅。然后向惊恐的族人冲去，利爪抬起，就要落下——

“铁山！”

一个少女的声音突然响起，清脆如冰凌碎裂。

霜狼的动作猛地一顿。

那是……雪莉的声音。铁山在意识深处挣扎，他想要停下来，想要恢复人形，但狼灵的力量太强了，像一座大山压在他身上，把他往深渊里拖，往黑暗里拽。

“铁山，你答应过要保护我的……”

少女的声音在颤抖，但那双眼睛却毫不退缩地盯着霜狼幽蓝的瞳孔，清澈如冰湖。

铁山感觉有什么东西在胸口炸开了。

他想起小时候和雪莉在雪地里追逐，雪花落在他们的头发上；想起第一次狩猎时她帮他包扎伤口，手指轻柔如羽毛；想起她说过的那句话——“铁山，你是我见过最勇敢的人。”

狼灵咆哮着，想要撕碎这个碍事的女孩。

但铁山怒吼着，死死压制住那股力量。

“滚开！”他咬着牙，几乎是从喉咙里挤出了这两个字，声音沙哑而痛苦。

霜狼的身躯开始缩小，狼毛褪去，骨骼收缩。铁山重重地摔在雪地上，浑身是血，赤裸的胸膛上刻满了狼灵留下的爪痕，纵横交错，触目惊心。

“铁山！”雪莉扑过来，用狼皮裹住他冰冷的身体，将他紧紧抱在怀里。

远处，赤焰站在烧焦的战马旁，看着这一幕，眼中闪过一丝复杂的情绪，有忌惮，有愤怒，还有一丝不易察觉的敬意。

“撤。”他挥手，“告诉主力部队，狼神之子比预想中更强，我们需要更多准备。”

铁山躺在雪地上，看着灰蒙蒙的天空，耳边回荡着父亲最后那句话。

月牙谷……

那里到底有什么？

他的手指在雪地上缓缓握紧，指甲刺破掌心，鲜血渗进冻土。体内的狼灵还在低吼，像一头困兽在咆哮。但这一次，铁山没有害怕。

因为他知道，有些答案，必须自己去寻找。

有些命运，必须自己去面对。

# 第3章 月牙谷的阴影

铁山醒来时，发现自己躺在一间陌生的木屋里。

屋顶漏下几缕晨光，空气中弥漫着草药与鹿皮的气息。他试图坐起，胸口却传来撕裂般的剧痛——那些狼灵留下的爪痕仍在渗血，皮肤下的每一寸肌肉都在痉挛，仿佛有无数根烧红的铁针在筋脉间游走。

“别动。”雪莉端着一碗热汤推门进来，“你昏迷了两天两夜。阿嬷说，要是再不醒，就得去找狼神萨满了。”

铁山接过汤碗，手指微微颤抖。他盯着碗里映出的自己——琥珀色的瞳孔深处，似乎有什么东西在游动，像一头困兽在黑暗中挣扎，随时要破笼而出。

“赤焰的部队呢？”

“撤了。”雪莉在床边坐下，声音压得很低，“但他们在山谷入口设了哨卡，说是要‘保护边境不受野兽侵扰’。其实是封锁了去月牙谷的路。”

铁山的手猛地收紧，陶碗发出碎裂的脆响。温热的汤水顺着指缝滴落，他却浑然不觉，只是盯着掌心那些被烫红的皮肤，仿佛疼痛来自另一个世界。

“父亲说，月牙谷里有答案。”

雪莉沉默片刻，从怀里取出一张泛黄的羊皮纸：“阿嬷让我把这个给你。她说，这是你父亲生前留下的，但一直没找到合适的机会交到你手上。”

铁山展开羊皮纸。上面的符文已经模糊不清，但能辨认出一幅粗略的地图——月牙谷的地形，标注着几个奇怪的符号。其中最醒目的，是一个倒置的狼头，双眼处画着两个圆环，像两轮血月悬在深渊之上。

“这是什么？”

“阿嬷说，这是‘狼神之瞳’。”雪莉的声音压得更低了，几乎被屋外的风声吞没，“传说中，狼神将自己的力量封印在月牙谷深处，只有通过试炼的人才能打开封印。但你父亲认为，那根本不是封印，而是——”

“是什么？”

“是牢笼。”雪莉的眼中闪过一丝恐惧，像冰面下的暗流，“他怀疑，狼神并非自愿放弃力量，而是被某种更强大的存在囚禁了。那座山谷里埋着的，不是祝福，而是诅咒。”

铁山的手指在羊皮纸上缓缓划过。触碰到那些符文时，指尖传来灼烧般的刺痛，像被火焰舔舐。他低头看去，那些符文竟在微微发光，像活过来一样沿着他的皮肤游走，钻进血管，在脉搏中跳动。

“雪莉，帮我准备干粮和武器。”

“你要去？”

“我必须去。”铁山站起身，胸口那些爪痕在晨光中泛着银光，像某种古老的烙印，“如果父亲说的是真的，那我欠他一个答案。”

雪莉张了张嘴，最终什么都没说。她从柜子里取出一把骨刀和一捆弓弦，放在铁山面前。刀刃上刻着细密的狼纹，在昏暗的光线中泛着冷冽的寒光。

“我跟你去。”

“不行。”

“我不是在请求你。”雪莉的声音突然变得坚定，像冻土下的根须，“我是狼族语学者。如果月牙谷里真的封印着什么，你需要有人解读那些符文。否则，你连入口都找不到。”
